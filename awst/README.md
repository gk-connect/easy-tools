
# AWS CLI HELPER TOOL

Shorthand commands to read aws resource data. 



## Installation

Install my-project with bash

```bash
  chmod +x insall.sh
  ./insall.sh
```
    
## Features

- ```--list-ec2``` #lists all ec2 instances



## Usage/Examples

```bash
awst --list-ec2 --region <region name> --profile <aws profile>
```
